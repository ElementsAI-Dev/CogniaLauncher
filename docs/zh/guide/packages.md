# 包管理

CogniaLauncher 集成 51+ Provider，提供统一的包搜索、安装和管理体验。

---

## 核心功能

### 包搜索

- **关键词搜索** — 在所有可用 Provider 中搜索
- **高级搜索** — 按 Provider、平台、类别过滤
- **搜索建议** — 输入时提供自动补全
- **包对比** — 并排比较多个包的信息

### 包安装

1. 搜索目标包
2. 查看包详情（版本、依赖、许可证、描述）
3. 选择版本（可选）
4. 点击 **安装**

支持的安装选项：

- 指定版本号
- 全局/项目级安装
- 强制重装

### 包管理

- **列出已安装** — 查看所有已安装的包
- **检查更新** — 检测可用更新
- **批量操作** — 批量安装/卸载/更新
- **版本锁定** — 固定包版本防止意外更新
- **版本回滚** — 回退到之前的版本

### 依赖解析

CogniaLauncher 使用 PubGrub 算法进行依赖解析：

- 自动解析包的依赖关系
- 检测版本冲突
- 生成安装计划

---

## 批量操作

批量操作支持对多个包同时执行操作：

```
批量安装 → 依赖解析 → 并行下载 → 顺序安装 → 结果报告
```

功能特性：

- **进度追踪** — 每个包独立的进度条
- **错误处理** — 单个包失败不影响其他包
- **安装历史** — 记录所有安装/卸载操作
