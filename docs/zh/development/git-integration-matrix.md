# Git 接入矩阵

本矩阵用于 Git 功能防回退基线，统一描述：
`src-tauri/src/commands/git.rs` 命令 -> `lib/tauri.ts` 封装 -> Hook API -> 页面/组件入口。

说明：
- 按流程分组，一行可包含多个命令。
- Hook API 来自 `hooks/use-git.ts`、`hooks/use-git-advanced.ts`、`hooks/use-git-lfs.ts`。
- UI 入口对应 `app/git/page.tsx` 及其卡片组件。

编辑器打开契约补充说明：
- `git_probe_editor_capability` 返回稳定字段：`available`、`reason`、`configPath`、`fallbackAvailable`。
- `git_open_config_in_editor` 现返回结构化 `EditorOpenActionResult`（`kind`、`reason`、`fallbackUsed`、`fallbackPath`），不再只返回文本。
- Git 页面与插件脚手架“在编辑器中打开”统一复用前端共享编排，确保失败回退行为一致。
