# 快捷键参考

CogniaLauncher 支持以下键盘快捷键。

---

## 全局快捷键

| 快捷键 | macOS | 功能 |
|--------|-------|------|
| ++ctrl+k++ | ++cmd+k++ | 打开命令面板 |
| ++ctrl+comma++ | ++cmd+comma++ | 打开设置 |

---

## 命令面板内

| 快捷键 | 功能 |
|--------|------|
| ++arrow-up++ / ++arrow-down++ | 上下选择 |
| ++enter++ | 执行选中项 |
| ++escape++ | 关闭命令面板 |
| 输入文字 | 模糊搜索过滤 |

---

## 导航

| 快捷键 | 功能 |
|--------|------|
| 侧边栏点击 | 页面导航 |
| 浏览器后退 | 返回上一页 |

---

## 窗口控制（桌面模式）

| 操作 | 描述 |
|------|------|
| 标题栏拖拽 | 移动窗口 |
| 标题栏双击 | 最大化/还原 |
| 最小化按钮 | 最小化到任务栏 |
| 关闭按钮 | 关闭窗口（最小化到托盘） |
