# Keyboard Shortcuts Reference

CogniaLauncher supports the following keyboard shortcuts.

---

## Global Shortcuts

| Shortcut | macOS | Function |
|----------|-------|----------|
| ++ctrl+k++ | ++cmd+k++ | Open command palette |
| ++ctrl+comma++ | ++cmd+comma++ | Open settings |

---

## Inside Command Palette

| Shortcut | Function |
|----------|----------|
| ++arrow-up++ / ++arrow-down++ | Navigate up/down |
| ++enter++ | Execute selected item |
| ++escape++ | Close command palette |
| Type text | Fuzzy search filter |

---

## Navigation

| Shortcut | Function |
|----------|----------|
| Sidebar click | Page navigation |
| Browser back | Return to previous page |

---

## Window Control (Desktop Mode)

| Action | Description |
|--------|-------------|
| Titlebar drag | Move window |
| Titlebar double-click | Maximize/restore |
| Minimize button | Minimize to taskbar |
| Close button | Close window (minimize to tray) |
