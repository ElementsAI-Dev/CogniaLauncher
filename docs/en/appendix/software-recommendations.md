### 文件管理器

#### Yazi

**闪电般快速的终端文件管理器**

Yazi 是一款使用 Rust 编写的现代化终端文件管理器，基于异步 I/O 架构设计。它专为追求效率的终端用户打造，提供了传统文件管理器无法比拟的速度和响应性。

- **极速性能**：基于异步 I/O，目录加载、文件扫描和预览渲染从不阻塞
- **图像预览**：支持在终端内直接预览图片
- **高度可定制**：支持主题、快捷键和插件扩展
- **跨平台**：支持 Linux、macOS 和 Windows
- **Vim 风格操作**：熟悉的键位绑定，无缝切换

> 🔗 GitHub: <https://github.com/sxyazi/yazi>

### 包管理器

#### uv

**极速 Python 包管理器**

uv 是由 Astral 公司开发的高性能 Python 包管理器，使用 Rust 编写，旨在成为 "Python 的 Cargo"。它比传统的 pip 快 10-100 倍，提供了统一的 Python 项目管理解决方案。

- **极致速度**：比 pip 快 10-100 倍，比 Poetry 快数十倍
- **统一工具链**：集成包管理、虚拟环境、Python 版本管理于一体
- **项目管理**：`uv run`、`uv lock`、`uv sync` 提供完整的项目工作流
- **工具管理**：`uvx` 命令可直接运行 CLI 工具，无需显式安装
- **Python 安装**：`uv python install` 自动安装和管理 Python 版本
- **跨平台锁文件**：生成可跨平台使用的标准化锁文件

```bash
# 安装
curl -LsSf https://astral.sh/uv/install.sh | sh

# 创建项目
uv init myproject && cd myproject
uv add requests numpy
uv run python main.py
```

> 🔗 官网: <https://astral.sh/uv>

#### pixi

**现代化跨语言包管理器**

pixi 是由 prefix.dev 开发的新一代包管理器，基于 conda 生态系统构建，但比 conda/mamba 更快、更现代。它专为科学计算和跨语言项目设计。

- **极速安装**：比 conda 快 10 倍，比 micromamba 快 4 倍
- **原生锁文件**：内置跨平台锁文件支持，确保可复现性
- **任务系统**：内置 task runner，支持跨平台任务定义
- **多语言支持**：不仅限于 Python，支持 R、C/C++、Rust 等
- **PyPI 集成**：深度集成 PyPI，可混合使用 conda 和 pip 包
- **工作区支持**：支持 monorepo 和多项目工作区

```bash
# 安装
curl -fsSL https://pixi.sh/install.sh | bash

# 从 conda 环境迁移
pixi init --import environment.yml

# 添加包并运行
pixi add python numpy pandas
pixi run python script.py
```

> 🔗 官网: <https://pixi.sh> | GitHub: <https://github.com/prefix-dev/pixi>

### 进程管理器

#### PM2

**Node.js 生产级进程管理器**

PM2 是一款功能强大的 Node.js 进程管理器，内置负载均衡器。它可以让你的应用 24/7 在线运行，并提供丰富的监控和管理功能。

- **永久在线**：应用崩溃后自动重启，支持开机自启
- **负载均衡**：集群模式充分利用多核 CPU
- **零停机重载**：热重载应用，不中断服务
- **实时监控**：CPU、内存使用率实时监控
- **日志管理**：集中管理应用日志
- **部署工作流**：内置部署系统，支持一键部署

```bash
# 安装
npm install pm2 -g

# 启动应用
pm2 start app.js
pm2 start "npm run start" --name my-app

# 集群模式（自动利用所有 CPU）
pm2 start app.js -i max

# 监控
pm2 monit
pm2 logs
```

> 🔗 官网: <https://pm2.keymetrics.io>

## 总结

本指南涵盖了开发者和效率工作者常用的工具软件。选择工具时，建议：

1. **按需选择**：不要追求大而全，选择真正需要的工具
2. **学习快捷键**：花时间学习工具的快捷键，事半功倍
3. **定期清理**：定期审视和清理不常用的工具
4. **关注更新**：保持工具更新，获取新功能和安全修复
