# Git Integration Matrix

This matrix is the regression baseline for Git feature wiring:
`src-tauri/src/commands/git.rs` command -> `lib/tauri.ts` wrapper -> hook API -> page/component entry.

Notes:
- Rows are grouped by workflow. A row can include multiple commands.
- Hook APIs are from `hooks/use-git.ts`, `hooks/use-git-advanced.ts`, and `hooks/use-git-lfs.ts`.
- UI entry is the main integration in `app/git/page.tsx` and its card components.
