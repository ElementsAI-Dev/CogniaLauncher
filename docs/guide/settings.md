# 设置与主题

CogniaLauncher 提供丰富的设置选项和主题定制功能。

---

## 设置页面

通过侧边栏底部的齿轮图标进入设置页面。设置按类别分组：

### 常规设置

- 语言切换（中/英）
- 并行下载数
- 版本解析策略
- 元数据缓存 TTL
- 自动更新元数据

### 网络设置

- 请求超时时间
- 失败重试次数
- HTTP/HTTPS 代理

### 路径设置

- CogniaLauncher 数据目录
- 缓存目录
- 环境安装目录
- 支持文件夹选择器（Tauri 模式）

### 缓存设置

- 最大缓存大小
- 缓存过期天数
- 自动清理开关及阈值
- 缓存监控间隔
- 外部缓存监控

### 外观设置

- 主题模式（浅色/深色/跟随系统）
- 强调色
- 图表颜色主题（6 种方案）

### 安全设置

- 允许 HTTP（仅开发环境）
- 证书验证
- 自签名证书

---

## 主题系统

### 颜色方案

基于 CSS 变量实现，定义在 `app/globals.css` 中。

支持的主题模式：

- **浅色模式** — 明亮背景
- **深色模式** — 暗色背景
- **跟随系统** — 自动匹配操作系统主题

### 强调色

可选择不同的强调色方案来个性化界面。

### 图表颜色主题

数据可视化图表支持 6 种配色方案：

| 主题 | 描述 |
|------|------|
| default | 默认配色 |
| vibrant | 鲜艳配色 |
| pastel | 柔和配色 |
| ocean | 海洋蓝配色 |
| sunset | 日落暖色 |
| monochrome | 单色配色 |

---

## 设置搜索

设置页面内置搜索功能：

- 按关键词过滤设置项
- 支持中英文搜索
- 自动定位到匹配的设置区域
- IntersectionObserver 追踪当前活动区域

---

## 快捷键

设置页面支持快捷键操作，详见 [快捷键参考](../reference/keyboard-shortcuts.md)。

---

## 系统信息

设置页面底部显示系统信息：

- 操作系统版本
- CPU 架构
- 内存大小
- CogniaLauncher 数据目录
- 系统运行时间

---

## 状态管理

设置数据存储在多个 Zustand Store 中：

| Store | 用途 |
|-------|------|
| `appearance.ts` | 主题、强调色、图表颜色主题 |
| `settings.ts` | 通用设置 |
| `window-state.ts` | 窗口状态 |

所有 Store 使用 `persist` 中间件实现 localStorage 持久化。
