# 命令面板

命令面板提供全局快捷搜索功能，快速访问 CogniaLauncher 的所有功能。

---

## 打开方式

- **键盘快捷键**：++ctrl+k++（macOS: ++cmd+k++）
- **侧边栏按钮**：点击搜索图标

---

## 功能

### 页面导航

输入页面名称快速跳转：

- 仪表板
- 环境管理
- 包管理
- 缓存管理
- 下载管理
- 设置
- 关于

### 操作搜索

搜索并执行常用操作：

- 安装环境版本
- 搜索包
- 清理缓存
- 切换主题
- 打开设置

### 搜索过滤

- 模糊匹配
- 按类别分组显示
- 键盘导航（上下箭头）
- 回车键执行选中项

---

## 实现

命令面板基于 [cmdk](https://cmdk.paco.me/) 构建：

- `components/command-palette.tsx` — 主组件
- 注册在 `app/layout.tsx` 中的全局键盘监听
- 支持自定义操作注册
