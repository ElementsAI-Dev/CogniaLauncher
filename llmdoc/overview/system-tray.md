# System Tray

## 1. Identity

- **What it is:** A multi-language system tray with dynamic icon states, autostart, notifications, and configurable click behavior.
- **Purpose:** Provide background presence, quick access to core functionality, and system-level integration.

## 2. High-Level Description

The system tray provides a persistent background interface with context menu actions (show/hide, settings, updates, logs, quit). Icon states reflect application status (normal, downloading, update available, error). Supports autostart management, system notifications, and configurable left-click behavior (toggle window, show menu, do nothing). Language syncs automatically with app locale.
